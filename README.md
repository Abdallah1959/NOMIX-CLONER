## 📌 About The Project
**NOMIX CLONER** is a professional, high-speed Discord utility tool designed for server owners and community managers. It provides seamless server cloning, channel scraping, and server wiping capabilities with an integrated GUI and top-tier security.

## ✨ Key Features
- **🛠️ Full Server Cloner:** Copy Roles, Categories, Channels, Emojis, and Stickers effortlessly.
- **🧹 Smart Wiper:** Selectively delete specific items or nuke the entire server.
- **📥 Message Scraper:** Backup entire channels into readable HTML, TXT, or JSON formats.
- **🔒 Highly Secure:** Powered by KeyAuth hardware and license security.
- **⚡ Cross-Platform:** Works smoothly on Windows, macOS, and Linux.

   - `macOS / Linux`: Download the specific `.zip` or `.tar.gz` file.
3. Run the application and enter your valid License Key!
